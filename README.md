# Java Tutorium

This repository contains the code examples, that are created during the tutorial lessons.
It is meant to be used as reference, when you want to write the explained example by your self or expand them.

### Get code via IntelliJ

File -> new -> Project from version control

url: https://github.com/JulianSobott/JavaTutorium.git

### Get code via `git`:

```shell script
git clone https://github.com/JulianSobott/JavaTutorium.git
```

### Get code via eclipse

1. File -> Import ->... | Git|Projects from Git -> next
2. Clone Uri -> next
3. URI: https://github.com/JulianSobott/JavaTutorium.git -> next
4. Import branches: 'master' others are optional -> next
5. Specify location where to save the repository -> next
6. Import existing projects -> Finish

For more information about using git in eclipse: https://www.vogella.com/tutorials/EclipseGit/article.html
